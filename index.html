How It Works

Our gateway acts as a proxy between your application and Virtusim API. We forward your request using Vercel's clean IP addresses (not blocked by Cloudflare).

Quick Start

// 1. Get your API key from @virtusim_bot
// 2. Replace YOUR_KEY below
// 3. Make request to our gateway

fetch('https://your-domain.vercel.app/api/balance?apikey=YOUR_KEY')
  .then(response => response.json())
  .then(data => console.log(data))
